# Virtual Studio Assembly

Overview
---

Developed a professional product presentation scene in Unreal Engine 5 to showcase a 3D model in a controlled studio environment. The project focuses on visual quality, lighting, camera presentation, user interaction, and cinematic rendering.

##### Completed Features

    * Studio Environment \& Lighting
    * Designed a dedicated studio scene for product visualization.
    * Configured a professional lighting setup to achieve clean illumination, realistic reflections, and balanced shadows.
    * Optimized the scene for Unreal Engine 5's Lumen global illumination.

##### Material Integration

    * Applied production-quality materials to the imported model.
    * Integrated the material workflow developed in the previous pipeline task for automated material assignment where applicable.

##### Cinematic Camera Setup

    * Configured a Cinematic Camera Actor for product presentation.
    * Tuned focal length, aperture, and focus settings to create a polished composition suitable for showcase renders.

##### Interactive Review Mode

    * Implemented a real-time review system using C++.
    * Added functionality to inspect the model by rotating the object and orbiting the camera during gameplay.

##### Cinematic Sequence

    * Created a short presentation sequence using Sequencer.
    * Produced a smooth camera animation to highlight the model from multiple viewing angles.

##### Final Rendering

    * Generated high-quality output using Unreal Engine's rendering pipeline.
    * Captured presentation-ready images and cinematic video suitable for portfolio or design review.

##### Project Organization

    * Organized assets, source code, materials, maps, and supporting resources using a structured folder hierarchy.
    * Maintained consistent naming conventions throughout the project for improved readability and maintainability.
